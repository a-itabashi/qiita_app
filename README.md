# post_app(like a qiita)

## 機能一覧
### ユーザー(users)
- Twitter認証のみ
- 退会機能

### 記事(articles)
- 記事一覧(投稿日時から降順/新しい順)
- ページネーション

### コメント(comments)
- コメント機能(Ajax)

### いいね機能(article_favorite)
- いいね機能(Ajax)
　※ 記事に紐づくいいね機能
